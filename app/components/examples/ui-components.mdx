### CSS Components

If only CSS is repeating but HTML structure changes, extract repeating classes in Enums.

constants.js

```js
const button = {
  PRIMARY: 'py-2 px-4 bg-primary-base text-white font-semibold rounded',
  SECONDARY: 'py-2 px-4 bg-secondary-base text-white font-semibold'
};
```

button.jsx

```jsx
<button type='button' className={`${button.PRIMARY} block mt-4`}>
  Button
</button>
```

Extracting css components in javascript instead of Tailwind CSS files makes them easier to test and keep track of changes.

### React Components

If both html and css are repeating together, prefer React Components.
